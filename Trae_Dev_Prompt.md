## 项目目标
构建一个面向泛用户的个人 Skill 展示网站，核心目标是沉淀个人品牌并展示个人能力。网站应支持后续持续新增 Skill 内容与模板下载能力。

## 技术要求
- 使用 Next.js
- 使用 Tailwind CSS
- 使用内容文件驱动管理 Skill 数据
- 目录结构清晰，方便后续 Claude 直接协助新增 Skill
- 首版以中文为主，但要预留英文扩展能力

## 页面要求
至少完成以下页面：
1. 首页
2. Skill 列表页
3. Skill 详情页
4. About 页面

## Skill 详情页必须包含字段
- 标题
- 简介
- 适用场景
- 使用方法
- 可下载模板

## 风格要求
参考 `https://www.vibevibe.cn/zh/` 的气质方向：
- 高级
- 现代
- 简洁
- 品牌感强
- 留白充足

注意：不是复制该网站，而是借鉴其视觉节奏、内容质感与品牌表达方式。

## 内容架构要求
建议采用以下目录：

```bash
/src
  /app
  /components
  /lib
  /types
/content
  /skills
    /zh
    /en
/public
  /downloads
```
