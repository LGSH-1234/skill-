# 初始需求沟通记录

- 日期：2026-04-02
- 项目类型：新项目启动
- 项目主题：个人对外展示 Skill 的网站

## 已确认需求

### 1. 项目目标
构建一个个人用于对外展示自身 Skill 的网站，并支持后续持续新增与上传 Skill 内容。

### 2. 目标用户
泛用户。

### 3. 核心目标
- 沉淀个人品牌
- 展示个人能力

### 4. Skill 类型
展示内容会比较全面，包括但不限于：
- AI 工作流
- 提示词能力
- 产品方案能力
- 开发能力
- 项目案例
- 可下载模板

### 5. Skill 详情页字段
每个 Skill 页面至少包含：
- 标题
- 简介
- 适用场景
- 使用方法
- 可下载模板

### 6. 后续维护方式
第一阶段采用“由 Claude 直接协助改代码并上传内容”的方式维护，而不是先做后台系统。

### 7. 产品阶段目标
第一阶段希望尽量完整一些，接近正式对外展示版本，而不是极简 MVP。

### 8. 视觉风格
参考网站：https://www.vibevibe.cn/zh/
目标风格为：
- 品牌感强
- 视觉简洁高级
- 现代化
- 具备一定作品集气质

### 9. 多语言策略
当前以中文为主，后续增加英文版本。

### 10. 技术方案偏好
采用推荐方案：Next.js + 内容文件驱动。

## 初步判断
这是一个“内容驱动型的个人品牌展示网站”，核心不是复杂交互，而是：
1. 品牌表达
2. Skill 结构化呈现
3. 内容的可持续更新
4. 为后续扩展双语和更多内容模块留好架构

## 下一步建议
1. 产出 MVP 产品方案与完整生命周期规划
2. 明确页面结构、内容模型和信息架构
3. 明确技术设计与内容目录规范
4. 生成开发指令，进入项目搭建阶段
