# 项目介绍与规划

## 项目定位
这是一个面向泛用户的个人 Skill 展示网站，核心目标是沉淀个人品牌并系统展示个人能力。网站既是个人能力名片，也是持续更新的数字作品集。

## 一句话价值主张
用一个风格统一、可持续更新的个人网站，清晰展示我的各类 Skill、使用场景与可下载模板，让外部用户快速理解我的能力边界与价值。

## 目标用户
- 泛用户
- 对 AI、效率工具、数字化能力感兴趣的人群
- 潜在合作方、关注者与口碑传播人群

## 核心目标
1. 沉淀个人品牌
2. 对外展示能力与方法论
3. 形成持续更新的 Skill 内容阵地
4. 为未来扩展英文内容、案例内容、联系方式留出空间

## 网站核心内容结构
- 首页：品牌形象、核心介绍、精选 Skill、整体能力概览
- Skill 列表页：展示所有 Skill 条目，可持续新增
- Skill 详情页：标题、简介、适用场景、使用方法、可下载模板
- 关于/个人介绍：建立个人品牌认知
- 后续扩展模块：英文版本、案例、联系方式、下载统计等

## 内容特点
- Skill 类型覆盖较广：AI 工作流、提示词、产品方案、开发能力、项目案例、可下载模板等
- 每个 Skill 既要能被快速浏览，也要支持深度阅读与模板下载
- 网站需要支持后续由 Claude 直接协助修改代码与新增内容

## 设计方向
参考 https://www.vibevibe.cn/zh/ 的整体气质与表达方向：
- 高级、现代、简洁
- 强视觉秩序感
- 偏品牌化与作品集感
- 页面留白充足，突出内容质感

## 版本策略
第一阶段不做过于简陋的 MVP，而是直接构建一个相对完整、接近正式对外版本的网站。

## 技术策略
采用：
- Next.js 作为前端框架
- 内容文件驱动方式管理 Skill 内容
- 优先保证后续新增 Skill 的低成本维护
- 预留双语扩展能力（当前中文为主，后续增加英文）

## 后续演进方向
- 中英双语切换
- 更丰富的 Skill 详情展示
- 下载追踪与转化分析
- 联系方式/合作入口
- 内容标签、筛选、搜索
- 案例与文章模块
