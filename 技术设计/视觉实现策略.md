# 视觉实现策略

## 一、视觉目标
网站整体气质参考 `https://www.vibevibe.cn/zh/`，但不复制其页面结构。视觉目标是：
- 高级
- 现代
- 简洁
- 品牌感强
- 具有作品集式表达

核心不是“炫”，而是“干净、稳、准”。

## 二、视觉原则

### 1. 大留白
通过足够的留白建立高级感，而不是依赖复杂装饰。

### 2. 强排版弱装饰
优先用：
- 标题层级
- 字重对比
- 行距
- 模块间距
来建立品质感。

避免：
- 大量渐变堆砌
- 花哨背景纹理
- 高频动画

### 3. 限制色彩数量
首版建议控制在：
- 1 个主品牌色
- 1 个中性色体系
- 1 个轻强调色（可选）

### 4. 内容即视觉
Skill 卡片和详情页内容本身就是视觉对象，组件应服务于内容，不要让装饰抢戏。

## 三、页面级视觉建议

### 首页
- Hero 区做大比例留白
- 标题字重明显
- 文案控制在短句，不宜过满
- 精选 Skill 区卡片样式统一，强调节奏感

### Skill 列表页
- 使用规整的卡片网格
- 卡片 hover 反馈轻微即可
- 分类标题与说明要克制

### Skill 详情页
- 顶部 Hero 区应更像品牌页面，而不是说明文档页
- 内容分区清晰，标题和正文的密度控制好
- 下载按钮要清楚，但不要做得廉价或过度营销

### About 页面
- 更偏品牌叙事
- 强调“我是谁、我擅长什么、我持续在做什么”

## 四、组件风格建议

### 导航
- 简洁横向布局
- 可使用细边框或轻背景区分
- 固定导航需谨慎，优先保证克制感

### 卡片
- 圆角适中
- 使用轻阴影或边框，而非厚重拟物
- hover 仅做轻微上浮、边框变化或背景变化

### 按钮
- 主按钮强调品牌色与清晰点击性
- 次按钮偏简洁线框或浅底
- 按钮文案短而明确

### 标签
- 尺寸偏小
- 作为辅助信息，不抢标题层级

## 五、文案气质建议
结合 vibevibe 的参考方向，文案应：
- 简短
- 克制
- 不浮夸
- 少用强销售感表达
- 更像专业创作者/产品人的自我表达

避免：
- 大量感叹句
- 空泛口号
- 过度营销化 CTA

## 六、动效建议
首版动效原则：轻、慢、少。

可使用：
- 模块淡入
- 卡片 hover 微动效
- 页面切换轻过渡

避免：
- 大面积视差
- 高频滚动特效
- 与内容无关的炫技动画

## 七、实现建议
技术实现上建议：
- 使用 Tailwind 建立统一 spacing / typography / color token
- 提前定义容器宽度、区块间距、标题层级规范
- 组件复用时优先统一节奏，不追求每块都不同

## 八、首版优先级
首版最需要先做好的不是“复杂设计”，而是以下四件事：
1. 首页 Hero 气质
2. 列表页卡片一致性
3. 详情页内容层级
4. 整站留白与排版秩序
